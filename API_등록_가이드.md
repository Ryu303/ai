# ARGOS 시스템 API 연동 등록 가이드 (초보자용)

본 설명서는 컴퓨터나 프로그래밍을 잘 모르는 일반 사용자도 쉽게 스마트홈 비서(ARGOS) 시스템에 연동 키(API)를 등록하고 사용할 수 있도록 단계별로 안내해 드립니다.

우측 하단의 **[환경설정 / API 연동]** 톱니바퀴 버튼을 누르신 뒤, 아래 안내에 따라 발급받으신 키를 입력하여 저장해 주세요.

---

## 1. SwitchBot API 토큰 발급 및 등록 방법
스마트 스위치, 스마트 커튼 등 SwitchBot 기기를 비서 시스템으로 켜고 끄기 위해 필요한 비밀번호입니다.

### 💡 발급 순서 (스마트폰 앱에서 진행)
1. 스마트폰에서 **SwitchBot 앱**을 켭니다.
2. 하단 메뉴에서 **[프로필]** (또는 **[마이페이지]**)을 누릅니다.
3. 메뉴 중 **[설정]** (톱니바퀴 모양 아이콘)으로 들어갑니다.
4. 화면 가장 아래에 있는 **[앱 버전]** 문구를 **연속으로 10번 빠르게** 터치합니다.
   - *팁: 스마트폰 개발자 모드를 켤 때처럼 톡톡톡 연타하시면 됩니다.*
5. 10번 누르면 숨겨져 있던 **[개발자 옵션]** 메뉴가 화면에 새로 나타납니다. 들어가 주세요.
6. 화면에 보이는 **[토큰(Token)]** 및 **[클라이언트 비밀값(Secret)]** 문자열을 복사합니다.
7. ARGOS 시스템의 **[환경설정 / API 연동]** 모달을 열고, **SwitchBot API Token** 입력란에 복사한 값을 붙여넣은 뒤 **[저장]** 버튼을 누릅니다.

---

## 2. Google OAuth Key (캘린더 / Gmail) 연동 방법
오늘 하루 구글 캘린더 일정을 읽어오고 안 읽은 메일을 브리핑해 주기 위해 구글 계정 권한을 승인하는 과정입니다.

### 💡 발급 순서 (컴퓨터 브라우저에서 진행)
1. **[구글 클라우드 콘솔](https://console.cloud.google.com/)** 사이트에 접속해 로그인합니다.
2. 화면 상단의 프로젝트 선택 창을 누르고 **[새 프로젝트]**를 클릭해 프로젝트를 하나 만듭니다 (이름은 `My Argos` 등 자유롭게 작성).
3. 왼쪽 메뉴에서 **[API 및 서비스]** -> **[라이브러리]**로 이동합니다.
4. 검색창에 다음 두 가지를 각각 검색하여 **[사용]** 버튼을 눌러 활성화합니다.
   - `Google Calendar API`
   - `Gmail API`
5. 다시 왼쪽 메뉴에서 **[OAuth 동의 화면]**으로 이동하여 **외부(External)**를 선택하고 만들기 버튼을 누릅니다. 앱 이름과 본인 이메일을 입력한 뒤 저장합니다.
6. 왼쪽 메뉴의 **[사용자 인증 정보]**로 이동하여 상단의 **[+ 사용자 인증 정보 만들기]** -> **[OAuth 클라이언트 ID]**를 선택합니다.
7. 애플리케이션 유형을 **웹 애플리케이션**으로 선택하고, 아래 항목을 입력합니다.
   - **승인된 리디렉션 URI**: `http://localhost:8000/api/v1/google/callback` 입력
8. 생성을 마치면 화면에 **[클라이언트 ID]**와 **[클라이언트 보안 비밀번호]**가 담긴 팝업창이 나타납니다.
9. **클라이언트 보안 비밀번호** 문자열을 통째로 복사하여 ARGOS 시스템의 **Google Calendar/Gmail OAuth Key** 란에 붙여넣고 **[저장]**을 누릅니다.

## 3. OpenWeather API Key (날씨 정보) 발급 및 등록 방법
외출 전 브리핑 시 오늘 동네의 정확한 온도와 기상 예보를 받아오기 위해 필요한 무료 날씨 키입니다.

### 💡 발급 순서 (컴퓨터 브라우저에서 진행)
1. **[OpenWeatherMap 홈페이지](https://openweathermap.org/)**에 접속하여 회원가입(Sign Up)을 합니다.
2. 회원 가입 및 이메일 인증을 완료한 후 로그인합니다.
3. 우측 상단 본인 계정 이름을 누르고 **[My API Keys]** 메뉴로 들어갑니다.
4. **Create Key** 영역에 이름(예: `Argos Weather`)을 적고 **[Generate]** 버튼을 누릅니다.
5. 생성된 표에서 `Key` 열에 있는 영어와 숫자가 무작위로 섞인 긴 문자열을 복사합니다.
6. ARGOS 시스템의 **OpenWeather API Key** 입력란에 복사한 키를 붙여넣고 **[저장]**을 누릅니다.

---

### 🔒 보안 안내 및 주의사항
- 등록해주신 모든 연동 비밀번호는 **대칭키 암호화(Fernet)** 처리를 거쳐 데이터베이스에 저장되므로, 해커가 침입하더라도 원본 키 값을 알아낼 수 없어 매우 안전합니다.
- 발급받으신 API 키는 타인이나 공공 장소에 노출되지 않도록 개인 메모장에 따로 적어두지 마시고 복사 즉시 등록해 주시기 바랍니다.
